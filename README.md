# Color Spot Challenge

## Project Description

Color Spot Challenge is a web-based visual puzzle game where players must identify the box with a slightly different color from the others.

As the game progresses, the number of boxes increases and their size decreases, making it more challenging to spot the odd one out. Each correct selection advances the player to the next level, improving focus and observation skills.

---

## Features

* Identify the odd-colored box
* Increasing difficulty with each level
* Dynamic grid size adjustment
* Level progression system
* Interactive and responsive UI

---

## Technologies Used

* HTML
* CSS
* JavaScript

---

##  How to Play

1. Start the game
2. Observe all the boxes carefully
3. Find the box with a slightly different color
4. Click on the correct box to move to the next level
5. Game ends if a wrong box is selected
